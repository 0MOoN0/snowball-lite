# Backend Docs

这里是后端长期说明入口。新的长期文档默认收口到这里，不再继续放在 `apps/backend/web/docs/` 根下。

## 当前正式入口

- [system-overview.md](system-overview.md)：后端目录职责、运行链路和协作约定
- [runtime-config.md](runtime-config.md)：当前有效的运行配置和环境变量口径
- [api-and-service-conventions.md](api-and-service-conventions.md)：接口和服务层写法
- [lite-mysql-matrix.md](lite-mysql-matrix.md)：lite 和传统 MySQL 的长期对照
